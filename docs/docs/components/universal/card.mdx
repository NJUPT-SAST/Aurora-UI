# Card Component

The Card component displays content within a styled card container.

## Props

| Property      | Description                                      | Type                                         | Default |
|---------------|--------------------------------------------------|----------------------------------------------|---------|
| `titleImageUrl`| The image URL for the card title.                | `string`                                     |         |
| `theme`       | The theme of the card (dark or light).           | `"dark" \| "light"`                         | `"light"`|
| `size`        | The size of the card (small, medium, large).     | `"small" \| "medium" \| "large"`            | `"medium"`|
| `shadow`      | The shadow style of the card.                    | `"regular" \| "small" \| "medium" \| "large" \| "extraLarge" \| "inner"` |         |
| `padding`     | The padding of the card.                         | `number`                                     | `16`    |
| `gap`         | The gap between the content and the header.      | `number`                                     | `16`    |
| `width`       | The width of the card.                           | `number`                                     | `300`   |

## Slots

| Name     | Description                                      |
|----------|--------------------------------------------------|
| `header` | The header content of the card.                  |
| `content`| The main content of the card.                    |
| `footer` | The footer content of the card.                  |

## Example

### Basic Example

```jsx
import { Card } from "@sast/ui-universal";

export default function Example() {
  return (
    <div style={{ margin: 10 }}>
      <Card
        theme="light"
        size="medium"
        padding={16}
        gap={16}
        width={300}
      >
        <div slot="header">Header Content</div>
        <div slot="content">Main Content</div>
        <div slot="footer">Footer Content</div>
      </Card>
    </div>
  );
}
```

### Login Card

```jsx
import { Card, Input, Button } from "@sast/ui-universal";

export default function Example() {
  return (
    <div style={{ margin: 30 }}>
      <Card
        theme="light"
        size="medium"
        padding={32}
        gap={16}
        width={350}
        shadow="large"
      >
        <div slot="header" style={{ display: 'flex',width: '100%', justifyContent: 'center'}}>
          <img
            src="/aurora-ui-logo-light.svg"
            alt="logo"
            style={{ width: 200, height: 100, margin: '0 auto' }}
          />
        </div>
        <div slot="content" style={{display: 'flex', flexDirection: 'column', width: '100%', justifyContent: 'center', gap: '10px'}}>
          <Input
            type="text"
            label="Username"
            fill
          />
          <Input
            type="password"
            label="Password"
            fill
          />
        </div>
        <div slot="footer" style={{marginTop: '20px', display: 'flex', gap: '10px'}}>
          <Button>
            Login
          </Button>
          <Button color='ghost'>
            Reset password
          </Button>
        </div>
      </Card>
    </div>
  );
}
```
